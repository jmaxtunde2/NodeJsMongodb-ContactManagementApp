# Contact Management App

A simple **Node.js** and **MongoDB** based Contact Management application that allows users to create, read, update, and delete contacts. This project demonstrates basic **CRUD operations** with a RESTful API and provides a foundation for building more advanced contact management systems.

---

## Features

- Add new contacts
- View all contacts
- Update existing contacts
- Delete contacts
- RESTful API endpoints
- MongoDB for data storage
- Input validation for contact details

---

## Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Environment:** dotenv for configuration

---

## Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn
- MongoDB (local installation or MongoDB Atlas)
